===========================================================
명령어란?
프로그램이다 ex) ipconfig, ls 등 os 탑재된 프로그램

PATH란?
디렉토리 주소를 전부 명시하지 않고 명령어로 구동하기 위해 미리 설정하는 것

명령어가 실행이 안된다?
=> 우선 해당 프로그램이 설치되어 있는지 확인

폴더 생성: mkdir [생성할 이름]
이동: cd [이동할 폴더 이름], cd - (뒤로가기), cd ..(한 단계 상위 디렉토리), cd [풀 경로]
현재 위치: pwd
파일 목록: ls (-al은 숨겨진 파일 전부 표시, -R 폴더와 그 내부 전부 한번에 확인, ll만 치면 ls -l)
폴더 삭제: rm -rf [폴더명] (폴더 안에 있는 내용과 함께 전부 삭제)
이름 변경: mv [전경로] [후경로] (이름 변경 및 이동)
복사: cp -r [전경로] [후경로] (복사)

**경로 입력방식 2가지**(중요)
절대경로: /root/c/a/e
상대경로: 절대경로를 제외한 모든 것이 상대경로(일반적으로 현재 경로를 기준으로 생각)
~은 윈도우로 치면 내문서 정도(/root와 동일?)

확장자란?
윈도우와 리눅스의 대부분의 파일은 텍스트형태, 실질적으로 중요한건 파일의 내용
확장자만 바꾼다고 파일자체의 성격이 변하지는 않음
확장자별로 내용의 형식이 조금씩 다름

여러개를 만들때는 공백으로 구분하면 됨
ex) mkdir a c (a,c두개 생성)

파일관련
touch [파일명]: 파일생성
cat, head, tail, more: 파일 내용확인, 나머지 이동, 복사 등은 같음
파란색(폴더), 흰색(파일)
ls -al로 조회했을때 맨앞이 d로 시작하면 directory, 아니면 파일

cat [경로]: 파일 내용 읽기********중요
head, tail등은 뒤에 숫자를 붙여서 특정 라인만 뽑아올 수 있음(그 외엔 잘 안씀)


편집:
vi 파일명(or nano 편집기)
3가지 모드 존재
VI 시작 => 명령모드 => 편집모드
VI 종료 <= Last Line 모드

	i(insert): 편집모드(커서 앞에서 편집 시작)
	a(insert): 편집모드(커서 뒤에서 편집 시작)
	esc: 명령모드로 나가기
	:w - 저장
	:q - 나가기
	:wq! - 저장하고 나가기(!는 강제 덮어쓰기)
커서이동
	G: 마지막줄
	gg: 첫 줄
	$: 커서 줄의 가장 오른쪽으로 이동
	0: 커서 줄의 가장 왼쪽으로 이동
	[라인번호]G: 해당라인으로 이동
	w: 한 단어 뒤
	b: 한 단어 앞
	
삭제
	dd: 한 줄 삭제
	d[커서이동]: 커서 이동만큼 삭제
	
복사
	yy: 한 줄 복사
	y[커서 이동]: 커서 이동만큼 복사
	p: 붙여넣기(커서 아래줄에)
	u: 되돌리기
	r: 해당 커서의 내용 수정
편집기 상태 변경
	:set nu 라인 넘버 표시
	:set nonu 라인 넘버 표시 안함

검색
	find [경로] [조건] [아규먼트] [행동]
	경로: 어디를 기준으로 검색?
	조건: 어떤 조건으로? ex) 이름이 ~~인
	아규먼트: 조건에 맞는 값
	행동: 결과 처리 어떻게?

--요약--
   1) 현재 작업 디렉토리 확인(탐색기에서 현재 위치가 표시되는 기능)
	pwd

  2) 디렉토리 이동하는 방법(폴더를 더블클릭해서 해당 폴더로 이동)
	cd  [이동할 경로]
	cd  -		이전 디렉토리로 이동
	cd  ..		한 단계 상위 디렉토리로 이동
	cd  /sjb/a	처음부터 입력(절대 경로)

 3) 디렉토리 내용 확인(탐색기에서 폴더 안에 있는 파일의 목록이 표시되는 기능)
	ls  -al

			a : 숨겨진 파일 표시
			l : 자세히 표시


2.1 디렉토리 작업
  1) 복사(폴더를 복사)
	cp  -r  [원본 경로]  [사본 경로]

		r : 폴더까지 복사

  2) 이동(폴더를 다른 곳으로 이동)
	mv  [원본 경로]  [이동할 경로]

  3) 이름 변경(폴더 이름 변경)
	mv  [원본 경로]  [바꿀 경로]

  4) 생성(새 폴더 생성)
	mkdir  [생성할 디렉토리 이름]

  5) 제거(폴더 삭제)
	rm  [삭제할 파일 or 디렉토리 경로]


2.2 파일 관련 명령어
  1) 파일 내용 확인(메모장 문서 파일 내용 확인)
	cat  [조회할 파일 경로]

  2) 복사(파일 복사)
	cp

  3) 이동(파일 옮기기)
	mv

  4) 이름 변경(파일 이름 변경)
	mv


  5) 생성(내용이 없음 빈 메모장 파일 생성)
	touch [생성할 파일 이름]


3. 검색(파일 또는 폴더 위치 검색)
	find  [경로]  -name  [이름]

4. 작업 관리자(ctrl+shift+esc로 작업관리자 실행하는 거)
	ps -ef

		pid : 컴퓨터가 실행된 프로그램을 구분하는 번호(매번 다르다)
		
	kill  -9  [pid]

5. 압축(압축 프로그램으로 파일 압축하거나 해제하는거)
	아카이브 파일 생성(tar는 크기측면의 압축이 아님)
	tar  cvf  [아카이브 파일이름]  [압축할 파일1]  [압축할 파일2]  ....

	아카이브 해제
	tar  xvf  [아카이브 파일이름]  

	압축
	gzip  [압축할 파일]

	압축 해제
	gunzip  [압축 파일 이름]

	압축 및 아카이브 한번에 해제
	tar  zxvf  [압축 및 아카이브 파일이름]
