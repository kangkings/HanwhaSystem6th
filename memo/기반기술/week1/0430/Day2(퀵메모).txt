리눅스 기본 명령어 추가

프로그램 설치(명령어를 찾을 수 없을 때)
yum install net-tools (인터넷이 될 때만 사용 가능)

IP 주소 확인
	ifconfig
	ip addr

IP주소 수동으로 설정
	vi /etc/sysconfig/network-scripts

	BOOTPROTO=static : 수동, dhcp: 자동
	ONBOOT=yes
	IPADDR=192.168.209.126
	NETMASK=255.255.255.0
	GATEWAY=

VMware로 IP주소 설정시 호스트 pc에 있는 공유기를 사용하면 안된다
=> VM에도 가상 공유기가 있음
virtual network editor VMnet8이 가상머신의 공유기

공유기는 보통 IP주소 마지막 클래스의 가장낮은 숫자인 1번을 사용한다
하지만 VM에서의 공유기는 2번까지 사용하기 때문에 3번째 클래스까지는
공유기와 같은 IP주소를 사용하고 마지막 클래스에서는 3번부터 사용해야한다.

서비스 프로그램 관리하는 법
리눅스는 설치하면 서비스로 등록이되기 때문에 서비스 관리 명령어로 실행한다
	systemctl start [서비스 프로그램 이름]
	systemctl restart [서비스 프로그램 이름]
	systemctl stop [서비스 프로그램 이름]
	systemctl status [서비스 프로그램 이름]

실행중인 포트 확인
	netstat -anlp
방화벽 off 명령어
	setenforce 0(로컬 방화벽)
	systemctl stop firewalld (네트워크 방화벽?)
	
CPU 가상화 설정 점검


바깥에서 공유기까진 접근 가능하지만 공유기에서 내부로 접근하려면
포트포워딩이 필요함
 VM은 NAT세팅에 포트포워딩 추가

내 컴퓨터의 아이피 :포워딩한 포트 => 공유기까지 접근 완료
=> NAT세팅에 해당 포트로 들어올 시 연결할 컴퓨터의 ip를 남겨뒀기 때문에
VMware안에 있는 가상 공유기를 거쳐서 실행중인 80번 포트 nginx까지 도달 가능
